Latihan Soal Transformasi Geometri

1. Tentukan persamaan bayangan garis 3x + 5y - 7 = 0 oleh translasi T = (2, -1)

Pembahasan:

Persamaan garis 3x + 5y - 7 = 0 ditranslasikan oleh T = (2, -1).

Substitusi: x = x' - 2 dan y = y' + 1

3(x' - 2) + 5(y' + 1) - 7 = 0

3x' - 6 + 5y' + 5 - 7 = 0

3x' + 5y' - 8 = 0

Jawaban: 3x + 5y - 8 = 0

2. Tentukan hasil bayangan titik A(3,5) oleh translasi T = (-2, 4)

Pembahasan:

A'(x', y') = (3 + (-2), (5 + 4)) = (1, 9)

Jawaban: A'(1, 9)

3. Diketahui titik P'(4, -12) adalah bayangan titik P oleh translasi T = (-9, 8). Koordinat titik P adalah?

Pembahasan:

P(x, y) = P'(x', y') - T = (4 - (-9), (-12 - 8) = (13, -20)

Jawaban: P(13, -20)

4. Titik A ditranslasikan oleh T = (6, -3) menghasilkan titik A'(4, -2). Koordinat titik A adalah?

Pembahasan:

A(x, y) = A'(x', y') - T = (4 - 6), (-2 - (-3)) = (-2, 1)

Jawaban: A(-2, 1)

5. Diketahui garis I: 4x - y + 2 = 0 ditranslasikan oleh T = (-2, 1). Maka bayangan garis I' adalah?

Pembahasan:

Substitusi: x = x' + 2 dan y = y' - 1

4(x' + 2) - (y' - 1) + 2 = 0

4x' + 8 - y' + 1 + 2 = 0

4x' - y' + 11 = 0

Jawaban: 4x - y + 11 = 0

6. Tentukan persamaan bayangan garis I: 2x - 3y + 12 = 0 jika ditranslasikan oleh T = (1, -2)

Pembahasan:

Substitusi: x = x' - 1 dan y = y' + 2

2(x' - 1) - 3(y' + 2) + 12 = 0

2x' - 2 - 3y' - 6 + 12 = 0

2x' - 3y' + 4 = 0

Jawaban: 2x - 3y + 4 = 0

7. Jika titik A(-4, -3) dicerminkan terhadap sumbu y maka bayangan titik A adalah?

Pembahasan:

Pencerminan terhadap sumbu y: (x, y) → (-x, y)

A' = (4, -3)

Jawaban: A'(4, -3)

8. Jika titik A(-4, -3) dicerminkan terhadap titik asal O(0, 0) maka bayangan titik A adalah?

Pembahasan:

Pencerminan terhadap titik asal: (x, y) → (-x, -y)

A' = (4, 3)

Jawaban: A'(4, 3)

9. Jika titik P(-5,4) dicerminkan terhadap garis y = x maka bayangan titik P adalah?

Pembahasan:

Pencerminan terhadap y = x: (x, y) → (y, x)

P' = (4, -5)

Jawaban: P'(4, -5)

10. Jika titik P(-5,4) dicerminkan terhadap garis y = -x maka bayangan titik P adalah?

Pembahasan:

Pencerminan terhadap y = -x: (x, y) → (-y, -x)

P' = (-4, 5)

Jawaban: P'(-4, 5)

11. Jika titik P(5, 2) dicerminkan terhadap garis x = 2 maka bayangan titik P adalah?

Pembahasan:

Pencerminan terhadap x = h: (x, y) → (2h - x, y)

P' = (4 - 5, 2) = (-1, 2)

Jawaban: P'(-1, 2)

12. Jika kurva y = x² + 3x -5 dicerminkan terhadap garis y = 2 maka hasil bayangan kurva adalah?

Pembahasan:

Pencerminan terhadap y = k: (x, y) → (x, 2k - y)

Substitusi: y = 4 - y'

4 - y' = x'² + 3x' - 5

y' = -x'² - 3x' + 9

Jawaban: y = -x² - 3x + 9

13. Tentukan bayangan titik C(3, 1) jika dirotasikan berlawanan arah jarum jam sebesar 90° dan berpusat di (2, 4)!

Pembahasan:

Rotasi 90° dengan pusat (a,b):

x' = a - (y - b)

y' = b + (x - a)

C' = (2 - (1 - 4), 4 + (3 - 2)) = (5, 5)

Jawaban: C'(5, 5)

14. Garis 3x - 4y + 12 = 0 dirotasikan sebesar 180° terhadap titik pusat (1, 2). Persamaan garis rotasi adalah?

Pembahasan:

Rotasi 180° dengan pusat (a,b):

x = 2a - x'

y = 2b - y'

Substitusi ke persamaan garis:

3(2 - x') - 4(4 - y') + 12 = 0

6 - 3x' - 16 + 4y' + 12 = 0

-3x' + 4y' + 2 = 0

Jawaban: -3x + 4y + 2 = 0

15. Tentukan bayangan titik A(-5, 2) setelah didilatasikan terhadap pusat (3, 4) dan faktor skala -3!

Pembahasan:

Dilatasi dengan pusat (a,b) dan skala k:

x' = a + k(x - a)

y' = b + k(y - b)

A' = (3 + (-3)(-5 - 3), 4 + (-3)(2 - 4)) = (3 + 24, 4 + 6) = (27, 10)

Jawaban: A'(27, 10)

16. Titik A(-2, -5) didilatasikan dengan faktor skala -2 terhadap titik pusat (0, 0). Tentukan hasil dilatasi pada titik A!

Pembahasan:

Dilatasi dengan pusat (0,0) dan skala k:

A' = (k*x, k*y) = (-2*-2, -2*-5) = (4, 10)

Jawaban: A'(4, 10)

17. Titik B didilatasikan dengan faktor skala -2 terhadap titik pusat (0, 0) menghasilkan titik B'(-4, 6). Koordinat titik B adalah?

Pembahasan:

B = (x'/k, y'/k) = (-4/-2, 6/-2) = (2, -3)

Jawaban: B(2, -3)
