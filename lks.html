A. Translasi (Pergeseran)

1. Diketahui titik A(3, -2). Tentukan koordinat bayangan titik A jika ditranslasikan oleh:

a) T(4, 5) → A'(..., ...)

b) T(-1, 3) → A'(..., ...)

Pembahasan:

a) A'(3+4, -2+5) = A'(7, 3)

b) A'(3+(-1), -2+3) = A'(2, 1)

2. Segitiga ABC dengan A(1,2), B(4,2), dan C(2,5) ditranslasikan oleh T(3,-1). Tentukan koordinat bayangan segitiga tersebut!

Pembahasan:

A'(1+3, 2+(-1)) = A'(4, 1)

B'(4+3, 2+(-1)) = B'(7, 1)

C'(2+3, 5+(-1)) = C'(5, 4)

B. Refleksi (Pencerminan)

3. Tentukan bayangan titik P(-3, 7) jika dicerminkan terhadap:

a) Sumbu x → P'(..., ...)

b) Sumbu y → P'(..., ...)

c) Garis y = x → P'(..., ...)

Pembahasan:

a) Refleksi sumbu x: (x,y) → (x,-y) → P'(-3, -7)

b) Refleksi sumbu y: (x,y) → (-x,y) → P'(3, 7)

c) Refleksi y = x: (x,y) → (y,x) → P'(7, -3)

4. Gambarlah segiempat ABCD dengan A(1,1), B(3,1), C(3,3), dan D(1,3) kemudian tentukan bayangannya jika dicerminkan terhadap garis x = 2!

Petunjuk: Buat sketsa pada kertas berpetak

C. Rotasi (Perputaran)

5. Tentukan bayangan titik Q(5, -1) jika dirotasikan:

a) 90° berlawanan jarum jam dengan pusat (0,0) → Q'(..., ...)

b) 180° dengan pusat (0,0) → Q'(..., ...)

Pembahasan:

a) Rotasi 90°: (x,y) → (-y,x) → Q'(1, 5)

b) Rotasi 180°: (x,y) → (-x,-y) → Q'(-5, 1)

D. Dilatasi (Penskalaan)

6. Titik R(6, 9) didilatasikan dengan faktor skala 2 dan pusat dilatasi di (0,0). Tentukan koordinat bayangan titik R!

Pembahasan:

Dilatasi dengan pusat (0,0) dan skala k: (x,y) → (kx, ky)

Jadi R'(2×6, 2×9) = R'(12, 18)

7. Segitiga DEF dengan D(2,1), E(5,1), dan F(2,4) didilatasikan dengan faktor skala 3 dan pusat (2,1). Tentukan koordinat bayangan segitiga tersebut!

E. Soal Aplikasi

8. Sebuah titik P(4,2) ditransformasikan dengan urutan berikut:

  1. Ditranslasikan oleh T(1, -3)
  2. Dicerminkan terhadap sumbu x
  3. Dirotasikan 90° searah jarum jam dengan pusat (0,0)

Tentukan koordinat titik akhir setelah semua transformasi tersebut!
